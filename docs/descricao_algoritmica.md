Descrição na Linguagem Algorítmica – Sistema Bontempo Contabilidade

Início do sistema.

Exibir tela de login.

Ler email e senha informados pelo usuário.

Se os dados estiverem corretos, então:
4.1 Redirecionar para o painel administrativo.
4.2 Exibir menu com as opções Cargo, Funcionário, Férias e Serviço.

Caso contrário:
5.1 Exibir mensagem de erro.
5.2 Retornar para tela de login.

No painel, ler opção selecionada pelo usuário.

Se a opção for "Cargo", então abrir submenu CRUD.

Se a operação selecionada for:

Cadastrar → Ler dados, validar e armazenar no banco.

Listar → Buscar todos os registros e exibir.

Editar → Buscar registro, exibir formulário, atualizar no banco.

Excluir → Remover registro selecionado.

Repetir o passo 8 para as entidades Funcionário, Férias e Serviço, seguindo os mesmos procedimentos CRUD.

Após executar qualquer operação, exibir mensagem de sucesso.

Perguntar se o usuário deseja continuar no sistema.

Se sim, retornar ao menu principal.

Se não, encerrar o sistema.

Fim.
